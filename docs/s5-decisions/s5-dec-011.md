# 의사결정 로그 DEC-011 ~

DEC-001~010 로그(s5-dec-001.md)에 이어지는 결정 기록.

---

## DEC-011 — kb10 이중 목적 전략
- 결정: 하나의 사이트에 언어권별 독립 목적 부여
- 영어(외국인 관광객): 보편 미용시술(써마지 등)로 트래픽·도메인 신뢰도 구축 (1순위)
- 한국어(국내 실수요자): 원장 전문영역(지방파괴주사, 지방흡입/이식 후 유착 제거 등) 니치 콘텐츠 (2순위, 후속 단계)
- 근거: 검색의도 분석상 한국어=가격·후기·비교 중심(플랫폼 장악), 영어=언어지원·신뢰성·총비용 중심(탐색 단계). 두 축은 경쟁하지 않음
- 공통 정체성: 서울의 미용시술을 신뢰할 수 있게 안내하는 사이트

## DEC-012 — 목적 기반 단일 언어 페이지 + 키워드 1:1 배정
- 결정: 각 페이지는 하나의 타깃 키워드를 전담하며 해당 검색시장에 맞는 단일 언어로만 생성. 기계번역판 금지(언어 중복 방지)
- 키워드 세분화: 같은 주제도 지역·범위로 세분화(예: 써마지-강남, 써마지-강남역, 써마지-논현역)하여 동일 키워드 경쟁 방지(카니발라이제이션 방지)
- 적용: 써마지-강남 파일럿은 영어 전용 확정, 한국어 버전 미생성

## DEC-013 — 관점 기반 독립 페이지 (번역판 배제)
- 결정: 같은 주제를 두 언어로 만들 경우에도 번역판이 아닌, 각 언어권 관점에서 필요 내용을 재구성한 독립 페이지로 작성. 언어 버전은 1:1 대응하지 않음
- 근거: 언어권별 검색의도 상이. DEC-007(중복 콘텐츠 회피)과 부합

## DEC-014 — 다국어 처리 정책 (UX + SEO) / DEC-004·009 부분 수정
- UX: 언어 전환 시 현재 페이지의 타 언어판이 아니라 해당 언어 홈으로 이동시키고 그 언어로 제공되는 콘텐츠 목록/옵션 제시. 대응 페이지 부재 시에도 막다른 길(404) 미발생
- SEO: 같은 콘텐츠의 언어 짝이 없으므로 페이지 간 hreflang 언어 짝 미연결. 언어 전환 링크(사용자용)와 hreflang(크롤러용)은 별개
- 상호참조: 본 결정은 DEC-004·DEC-009를 부분 수정함. 초기 양 언어 대칭 전제가 데이터·전략 검토로 변경됨

## DEC-015 — 의사결정 로그 파일 명명·운영 규칙
- 명명: 파일명 = 그 파일이 시작되는 decision 번호(3자리, 예: s5-dec-001.md, s5-dec-011.md). 마지막 번호는 파일명에 넣지 않음(항목 추가 시 파일명 불변)
- 운영: 기록량이 적으면 마지막 파일 하단에 append, 많으면 새 파일 생성 후 시작 번호로 명명
- 배경: 기존 s5-01-dec001-008.md는 파일명에 범위(개수)를 넣어 항목 추가마다 개명 필요 문제 발생 → s5-dec-001.md로 개명하며 규칙 정립
