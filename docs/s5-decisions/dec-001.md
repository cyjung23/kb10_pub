# 의사결정 로그 DEC-001 ~

프로젝트 kickoff 이전 논의에서 확정된 핵심 결정 기록. 모든 후속 작업의 단일 기준(single source of truth).

---

## DEC-001 — 도메인 확정
- 결정: kbeauty10.com 을 메인 도메인으로 확정 (가비아 구매 완료)
- 근거: kbeauty 키워드 보존 + 10 리스티클 컨셉 반영. 대안 도메인보다 SEO·시각적 주목도 우위
- 방어: 필요 시 kbeautyten.com, 10kbeauty.com 추가 확보 후 리다이렉트

## DEC-002 — 서비스 주제
- 결정: K-beauty 미용의료 정보 큐레이션 (피부과 중심)
- 근거: 소규모 팀 자원 한계상 토픽 authority 확보에 유리. 피부과가 외국인 환자 성장률 최상위(+86.2%)

## DEC-003 — 핵심 타깃
- 결정: 고객군 B(관광 중 피부과 시술 관광객)를 주축. 고객군 A(성형·수술 목적)는 보조
- 근거: B는 시장 크고 성장 빠름, 짧은 다운타임, 웹 검색·비교 행동이 사이트 모델과 일치

## DEC-004 — 언어 전략
- 결정: English 기본(defaultLocale=en) + 한국어. 하위폴더(/en/, /ko/) + hreflang. 기계번역 금지
- 근거: 얇은 다국어 페이지 회피, 언어당 페이지 품질 우선. 루트는 en 노출 + x-default, 강제 리다이렉트 없이 언어 전환 배너 제공
- SCP 교훈: defaultLocale 변경 이슈 회피 위해 처음부터 en 기본으로 시작
- 수정: DEC-014에 의해 부분 수정됨 (언어별 1:1 대응 페이지가 없으므로 hreflang 언어 짝 미적용)

## DEC-005 — 제목 규칙
- 결정: 10 [시술] Clinics in [지역], Seoul (2026 Guide) 스타일 채택
- 규칙: 연도(2026)는 제목·본문·meta에만 넣고 URL에는 넣지 않음. 연도 갱신 시 콘텐츠도 함께 갱신
- 근거: 정보성 표현으로 Best/Top 광고성 표현 회피(의료법), freshness 신호 확보

## DEC-006 — URL 구조
- 결정: 루트 아래 평면(플랫) 구조. 계층 폴더 미사용
- 조합 페이지: 시술-지역 순 (/rejuran-gangnam/, /botox-myeongdong/)
- 단독 시술: /rejuran/, /botox/ / 증상: /darkspots/, /acne-scars/ / 병원 상세: /clinic/[name]/
- 색인 페이지: /treatments/, /areas/, /symptoms/ (링크망 형성용)
- 근거: 신규·소규모 사이트에 평면 구조 유리. 단 내부 링크망 필수(고립 페이지 방지)

## DEC-007 — 법적 기준선 (SCP 승계)
- 결정: 4원칙 준수
  1. 병원으로부터 금전 미수취 (무대가)
  2. 예약·상담·송객 기능 없음 (무중계)
  3. 순위·비교·과장·전후사진·후기 배제 (중립 표현)
  4. SCP DB 원문 반드시 리라이팅 (중복 콘텐츠 회피)
- 근거: 의료법 제56조 리스크 회피. 무대가+무중계가 핵심 방어선
- 미결: 병원 카드 가격표기 수준(지역평균 vs 개별) 원장 의견 대기. 의료광고 전문 변호사 자문 권고

## DEC-008 — 저장소·작업 환경
- 결정: cyjung23/kb10 (Private, 소스코드) / cyjung23/kb10_pub (Public, 문서+README)
- 작업: seoulcp Codespaces 터미널, heredoc 기반 파일 작성
- 규칙: 절대경로 사용, 3단계 안전 편집, git 명령 분리

## DEC-009 — 루트 언어 자동감지 비활성화
- 결정: next-intl localeDetection false. 루트(/) 접속 시 항상 defaultLocale(en)로 리다이렉트
- 근거: DEC-004 방침 일치. Google 권장(자동감지 강제 리다이렉트 회피). 주 타깃이 영어권 관광객
- 보충: 검색 유입은 개별 언어 페이지(/en/, /ko/)가 직접 노출되므로 루트 리다이렉트와 무관
- 수정: DEC-014에 의해 부분 수정됨 (대응 언어 페이지가 없는 경우 hreflang 언어 짝을 걸지 않음)

## DEC-010 — 가비아 도메인 소유자명 오타 수정 보류
- 결정: kbeauty10.com 소유자명 한글 오타 수정을 사이트 완성·트래픽 발생 시점까지 보류
- 근거: 가비아 정책상 오타 정정은 소유권 이전 절차로만 가능(비용 22,000원). 운영에는 영향 없음
- 조건: 소유권 증빙 필요 상황 발생 전 처리 권장. 착수 시 동일인 단순 오타 무료 정정 가능 여부 재문의
